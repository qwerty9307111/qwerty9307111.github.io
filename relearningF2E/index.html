# CSS语法：除了属性和选择器，你还需要知道这些带@的规则

CSS 的顶层样式表由两种规则组成的规则列表构成，一种被称为 at-rule，也就是 at 规则，另一种是 qualified rule，也就是普通规则。

  • @charset
  • @import
  • @media
  • @page
  • @counter-style
  • @keyframes
  • @fontface
  • @supports
  • @namespace

# @charset

@charset 用于提示 CSS 文件使用的字符编码方式，它如果被使用，必须出现在最前面。这个规则只在给出语法解析阶段前使用，并不影响页面上的展示效果。

@charset "utf-8";
1

# @import

@import 用于引入一个 CSS 文件，除了 @charset 规则不会被引入，@import 可以引入另一个文件的全部内容。

@import "mystyle.css";
@import url("mystyle.css");

# @media

@media 对设备的类型进行一些判断。在 media 的区块内，是普通规则列表

@media print {
  body { font-size: 10pt; }
}

# @page

@page 用于分页媒体访问网页时的表现设置，页面是一种特殊的盒模型结构，出了页面本身，还可以设置它周围的盒。

@page {
  size: 8.5in 11in;
  margin: 10%;
  @top-left {
    content: "Hamlet";
  }
  @top-right {
    content: "Page " counter(page);
  }
}

# CSS 函数

  • 图片

    • filter
      • blur(radius) // 将高斯模糊应用于输出图片
      • brightness(amount) // 让图像更明亮或更暗淡
      • contrast(amount) // 增加或减少图像的对比度
      • drop-shadow(offset-x offset-y blur-radius color) // 在图像后方应用投影
      • grayscale(amount) // 将图像转为灰度图
      • hue-rotate(angle) // 改变图像的整体色调
      • invert(amount) // 反转图像颜色
      • opacity(amount) // 改变图像透明度
      • saturate(amount) // 超饱和或去饱和输入的图像
      • sepia(amount) // 将图像转为棕褐色
    • cross-fade() // 用于以定义的透明度混合两个或多个图像。

  • 图形绘制

    • conic-gradient() // 圆锥渐变
    • linear-gradient() // 线性渐变
    • repeating-linear-gradient() // 重复线性渐变
    • radial-gradient() // 径向渐变
    • repeating-radial-gradient() // 重复径向渐变

  • 布局

    • calc() // 属性计算
    • clamp() // 边界
    • fit-content() // 自适应边界
    • max() // 最大值
    • min() // 最小值
    • minmax() // 区间值（与 css 网格一起使用）
    • repeat() // 重复

  • 变形/动画

    • transform // 变换
      • matrix() // 矩阵
      • matrix3d() // 3D 矩阵
      • perspective() // 透视
      • translate() // 位移
      • translate3d()
      • translateX()
      • translateY()
      • translateZ()
      • scale() // 缩放
      • scale3d()
      • scaleX()
      • scaleY()
      • scaleZ()
      • rotate() // 旋转
      • rotate3d()
      • rotateX()
      • rotateY()
      • rotateZ()
      • skew() // 扭曲
      • skewX()
      • skewY()

  • 环境与元素

    • var() // 声明
    • env() // 环境
    • attr() // 属性
